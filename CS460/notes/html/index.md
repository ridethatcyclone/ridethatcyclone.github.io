### A simple HTML layout
```
<!DOCTYPE html>
<html>
    <head><title>Title</title></head>
    <body>
        Content goes here
    </body>
</html>
```

### Cheatsheet of useful HTML tags

Tag | Usage
----|------
<p> | Creates a paragraph
<img src="url"> | Inserts an image
<q> | Signifies a quotation
<blockquote> | Creates a blockquote
